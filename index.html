JAVASCRIPT FLASHCARDS

Functions

a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).

Data Types

specify what kind of data can be stored and manipulated within a program. ... String, Number, and Boolean are primitive data types. Object, Array, and Function (which are all types of objects) are composite data types.

Control Flow

is the order in which the computer executes statements in a script. A typical script in JavaScript includes many control structures, including conditionals, loops and functions. A script may be set to execute when events occur.

DOM

The Document Object Model (DOM) is a programming interface for HTM documents so that programs can change the document structure, style, and content.
